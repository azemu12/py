# 抖音视频下载工具

## 项目介绍

这是一个用于批量下载抖音用户视频的工具，支持根据用户ID列表自动获取用户信息并下载其作品。

## 功能特性

- 根据用户ID获取用户链接
- 批量获取用户作品信息
- 视频过滤（时长8秒-5分钟，分辨率1080P及以上）
- 自动下载视频到本地
- 记录下载进度和失败用户
- 支持重试机制，提高下载成功率


## 安装说明

### 1. 克隆项目

```bash
git clone <项目地址>
cd <项目目录>
```

### 2. 安装依赖

#### Python依赖

```bash
pip install -r requirements.txt
```

#### Node.js依赖（可选，用于部分签名功能）

```bash
npm install
```

在 `.env` 文件中配置必要的环境变量：

```
# .env 文件内容
# 抖音Cookie信息
DOUYIN_COOKIE=<你的抖音Cookie>
```

## 使用方法

### 1. 准备用户ID列表

在 `json_good` 目录下创建一个JSON文件，格式如下：

```json
[
    {"user_id": "用户ID1"},
    {"user_id": "用户ID2"},
    ...
]
```

### 2. 运行主程序

修改 `DY_main.py` 中的配置参数：

```python
# ===== 配置 =====
date_time = "20260401"  # 日期标识
json_file = f"json_good/{date_time}_user_id_list.json"  # 用户列表文件
save_dir = r"G:\DY_video_download"  # 保存路径
work_total_num = 10000  # 每个用户要爬取的作品数量
```

然后运行：

```bash
python DY_main.py
```

## 核心功能说明

### 1. 获取用户链接

通过 `DouyinAPI.get_user_link_by_user_id()` 方法根据用户ID获取用户主页链接。

### 2. 获取用户作品信息

通过 `DouyinAPI.get_user_all_work_info()` 方法获取用户的所有作品信息，并进行过滤：
- 时长过滤：只保留8秒到5分钟的视频
- 分辨率过滤：只保留1080P及以上的视频

### 3. 下载视频

通过 `download_work()` 方法下载视频到本地，支持：
- 断点续传
- 下载进度显示
- 失败重试
- 去重（避免重复下载）

## 注意事项

1. 请遵守抖音平台的使用规则，不要滥用此工具
2. 下载视频时请注意网络带宽和存储空间
3. 频繁请求可能会导致IP被封禁，建议合理控制请求频率
4. 本工具仅用于学习和研究目的

## 常见问题

### 1. 下载失败怎么办？

- 检查网络连接
- 检查Cookie是否有效
- 查看日志文件中的错误信息
- 尝试减少并发下载数量

### 2. 如何获取抖音Cookie？

1. 打开浏览器，登录抖音网页版
2. 打开开发者工具（F12）
3. 切换到Network选项卡
4. 刷新页面，找到任意一个抖音API请求
5. 在Request Headers中复制完整的Cookie值

### 3. 如何提高下载速度？

- 确保网络连接稳定
- 选择合适的时间段进行下载（避开网络高峰期）
- 适当调整重试次数和延迟时间
